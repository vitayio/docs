title: "The reference link isn't working"

---

If clicking the link in your reference request email leads to an error — or the reference ID you were given isn't being accepted — here's what's likely happening and how to fix it.

## Common reasons a link stops working

**The link has already been used**

Once a reference form is submitted, the link is locked. If you see a message saying the reference is already complete, your submission went through successfully and there's nothing else to do.

**The link has expired**

Reference request links can expire after a period of time. If the request is no longer active, only the recruiter can reissue it.

**The link was copied incorrectly**

If you copied the link manually from the email (rather than clicking it), check that the full URL was included with no missing characters or extra spaces at the end.

**Your work network is blocking the link**

Some company networks block external links in emails. Try opening the link on your personal device or using your phone's mobile data instead.

## What to do if none of the above helps

<Steps>
  <Step title="Find the recruiter's contact details">
    Open your original reference request email. The recruiter's name is mentioned in the message, and replying to that email will reach them directly.
  </Step>
  <Step title="Ask them to reissue the link">
    Let the recruiter know the link isn't working. They can check the status and send you a fresh request if needed.
  </Step>
</Steps>

<Note>
  Only the recruiter can reissue a reference link. There is no self-serve way to generate a new one.
</Note>
